# Visualizing-the-History-of-Nobel-Prize-Winners


🏅 Nobel Prize Data Analysis

📌 Project Overview

The Nobel Prize is one of the most prestigious international awards, established by Alfred Nobel (1833–1896) and first awarded in 1901. Each year, prizes are awarded for outstanding contributions in:
	•	Physics
	•	Chemistry
	•	Physiology or Medicine
	•	Literature
	•	Peace
	•	Economic Sciences

In this project, we analyze Nobel Prize laureate data from 1901 to 2023 to uncover patterns related to gender, nationality, and repeat winners.

The dataset used in this analysis comes from the Nobel Prize API and is provided in the file nobel.csv.

⸻

📊 Objectives

This project explores the Nobel Prize dataset to answer the following questions:

1️⃣ Most Common Gender and Birth Country

Identify:
	•	The most frequently awarded gender
	•	The most common birth country of laureates

Stored as:

top_gender
top_country


⸻

2️⃣ Decade with the Highest Proportion of US-Born Winners

Determine which decade had the highest ratio of US-born Nobel Prize winners relative to the total number of winners.

Stored as:

max_decade_usa


⸻

3️⃣ Decade and Category with Highest Female Representation

Identify the decade and category combination with the highest proportion of female Nobel Prize laureates.

Stored as a dictionary:

max_female_dict

Example:

{1990: "Literature"}


⸻

4️⃣ First Woman to Win a Nobel Prize

Determine:
	•	The first woman to receive a Nobel Prize
	•	The category in which she won

Stored as:

first_woman_name
first_woman_category


⸻

5️⃣ Repeat Nobel Prize Winners

Identify individuals or organizations that have won the Nobel Prize more than once.

Stored as a list:

repeat_list
